Dr. Kover VPN - Windows Quick Reference Guide
==============================================

LOCATION: C:\VPN-Management\

BASIC COMMANDS:
===============

1. Open PowerShell as Administrator
2. Enable script execution (one-time setup):
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
3. Navigate to management directory:
   cd C:\VPN-Management

Add a new device:
.\Add-Peer.ps1 -DeviceName [DeviceName]
Example: .\Add-Peer.ps1 -DeviceName "iPad"

Check VPN status:
.\VPN-Status.ps1

List configuration files:
Get-ChildItem *.conf

AFTER ADDING A DEVICE:
======================
1. Restart WireGuard tunnel in the WireGuard app
2. Copy the DrKover-[DeviceName].conf file to the device
3. Import the .conf file into WireGuard app on the device
4. Connect and test

TROUBLESHOOTING:
================

- Check server configuration exists
- Test connectivity after adding devices
